This page provides an overview of available API endpoints in Akari's platform. All endpoints use the base URL: `https://akari-psi.vercel.app/api`.

    ## Manga API

    Access manga content, search for titles, and browse collections.

    | Endpoint | Description |
    |----------|-------------|
    | `GET /manga/[id]` | Get manga details by ID |
    | `GET /manga/[id]/[subId]` | Get specific chapter/volume |
    | `GET /manga-list/latest` | List recently updated manga |
    | `GET /manga-list/popular` | List popular manga |
    | `GET /author/[id]` | List manga by author |
    | `GET /genre/[id]` | Browse manga by genre |
    | `GET /search/simple` | Search for manga titles |

  </TabContent>

  <TabContent id="account">
    ## Account API

    Manage user accounts, reading history, and bookmarks.

    | Endpoint | Method | Description |
    |----------|--------|-------------|
    | `/account/reading/history` | GET | Get reading history |
    | `/account/reading/stats` | GET | Get reading statistics |
    | `/bookmarks` | GET | List all bookmarks |
    | `/bookmarks/add` | POST | Add a bookmark |
    | `/bookmarks/delete` | DELETE | Remove a bookmark |
    | `/bookmarks/isbookmarked` | GET | Check bookmark status |
    | `/bookmarks/notification` | GET | Get bookmark notifications |
    | `/bookmarks/update` | PUT | Update a bookmark |
    | `/login` | GET/POST | Authenticate user |
    | `/logout` | GET | End user session |
    | `/logout/mal` | GET | End MAL session |
    | `/register` | GET/POST | Create new account |

  </TabContent>

  <TabContent id="mal">
    ## MyAnimeList API

    Integrate with MyAnimeList for tracking and synchronization.

    | Endpoint | Method | Description |
    |----------|--------|-------------|
    | `/manga/malsync/get` | GET | Get MAL sync info |
    | `/auth/token/get` | GET | Get MAL auth token |
    | `/auth/token/refresh` | GET | Refresh MAL token |
    | `/auth/token` | POST | Submit auth token |
    | `/mal/[id]` | GET | Get manga from MAL |
    | `/mal/check-vote` | GET | Check manga rating |
    | `/mal/isLoggedIn` | GET | Check MAL login status |
    | `/mal/me` | GET | Get MAL user info |
    | `/mal/me/mangalist` | GET | Get user's MAL list |
    | `/mal/report` | POST | Submit report to MAL |
    | `/mal/search` | GET | Search MAL manga |
    | `/mal/vote` | POST | Rate a manga |

  </TabContent>
</Tabs>

## Response Format

API responses follow a consistent JSON structure:

```json
// Success
{
  "success": true,
  "data": { /* Response data */ },
  "performance": {  /* Performance data */ }
}

// Error
{
  "success": false,
  "error": {
    "code": 404,
    "message": "Not found"
  }
}
```

For more detailed documentation on specific endpoints, please refer to the individual API reference pages.
