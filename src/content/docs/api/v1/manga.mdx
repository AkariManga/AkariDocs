## GET /manga/[id]

Get manga details by ID

### Response

```typescript
{
    mangaId: string | null;
    identifier: string;
    imageUrl: string;
    name: string;
    alternativeNames: string[];
    authors: string[];
    status: string;
    updated: string;
    view: string;
    score: number;
    genres: string[];
    description: string;
    chapterList: MangaChapter[];
    malData: null | {
        id: string;
        created_at: string;
        updated_at: string;
        image: string;
        description: string | null;
        score: number | null;
        mal_id: number;
        ani_id: number | null;
        type: MangaType;
    };
}
```

## GET /manga/[id]/[subId]

Get a manga chapter by its sub ID

### Response

```typescript
{
    id: string;
    title: string;
    chapter: string;
    number: string;
    chapters: { value: string; label: string }[];
    pages: number;
    parentId: string;
    nextChapter: string;
    lastChapter: string;
    images: string[];
    mangaId: string | null;
    chapterId: string | null;
    type: "Manga" | "Manhwa" | "Manhua" | null;
    malId: number | null;
    malImage: string | null;
}
```

## GET /manga/list/latest

List recently updated manga

### Query Parameters

| Parameter | Type   | Required | Description                                 |
| --------- | ------ | -------- | ------------------------------------------- |
| `page`    | number | No       | The page number for pagination (default: 1) |

### Response

```typescript
{
    mangaList: Array<{
        id: string;
        image: string;
        title: string;
        chapter: string;
        chapterUrl: string;
        description: string;
        views: string;
    }>;
    popular: Array<{
        id: string;
        image: string;
        title: string;
        chapter: string;
        chapterUrl: string;
    }>;
    metaData: {
        totalStories: number;
        totalPages: number;
    }
}
```

## GET /search

### Query Parameters

| Parameter | Type   | Required | Description                                 |
| --------- | ------ | -------- | ------------------------------------------- |
| `q`       | string | Yes      | The search query string                     |
| `p`       | number | No       | The page number for pagination (default: 1) |

### Response

```typescript
Array<{
    id: string;
    image: string;
    title: string;
    chapter: string;
    chapterUrl: string;
    author: string;
    date: string;
    views: string;
}>;
```
